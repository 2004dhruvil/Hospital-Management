# 🏥 Hospital Management System

A complete web-based **Hospital Management System** developed using **Python Django**. This project helps manage hospital operations such as doctor management, patient records, appointments, and appointment status handling.

---

# 📌 Features

✅ Doctor Management
✅ Patient Management
✅ Appointment Booking
✅ Appointment Approval/Rejection
✅ Doctor Availability Management
✅ Contact & About Pages
✅ Admin Panel Support
✅ Responsive User Interface
✅ Notification System

---

# ⚙️ Installation & Setup

## Step 1: Clone Repository

```bash


## Step 2: Move into Project Directory

```bash
cd your-repository-name
```

## Step 3: Create Virtual Environment

```bash
python -m venv venv
```

## Step 4: Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

## Step 5: Install Dependencies

```bash
pip install django
```

OR

```bash
pip install -r requirements.txt
```

## Step 6: Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

## Step 7: Run the Project

```bash
python manage.py runserver
```

# ▶️ Open in Browser

After running the server, open:

# 🔑 Admin Panel Access

To create admin account:

```bash
python manage.py createsuperuser
```

Then login at:

```bash
http://127.0.0.1:8000/admin/
```
